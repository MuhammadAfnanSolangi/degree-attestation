# Degree Attestation System

A blockchain-based system for degree attestation.

## Project Structure

```
degree-attestation/
├── backend/
│   ├── server.js
│   ├── blockchain.js
│   ├── contracts.js
│   ├── accessControl.js
│   ├── seed.js
│   └── package.json
└── frontend/
    ├── index.html
    ├── styles.css
    └── app.js
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

Open `frontend/index.html` in your browser.
